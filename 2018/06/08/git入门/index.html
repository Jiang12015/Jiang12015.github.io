git入门

git基本操作

1.git init
git init命令创建一个空的Git仓库或重新初始化一个现有仓库.在路径中建立一个.git目录。

2.git add
git add命令将文件内容添加到索引(将修改添加到暂存区)。也就是将要提交的文件的信息添加到索引库中。

3.git commit -v
git commit -v命令提交暂存区到仓库区并将显示所有更改过的信息。
